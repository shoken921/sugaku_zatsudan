# 雑談

昔ふと思い付いて証明した定理について紹介しようと思う。
主張はこうだ。

- $m,n\in\mathbb{N}$として、素体$\mathbb{F}_p$に$x^n=1$の解がちょうど$m$個存在するような素数$p$が存在する必要十分条件は、$m$と$n$の偶奇が一致し、かつ$m$が$n$の約数であることである。

証明の前に証明に必要な物について述べる。まず群論の基礎的な知識は前提として、中国剰余定理と原始根の存在定理とそれから大道具としてディリクレの算術級数定理を使う。俺はディリクレの算術級数定理の証明なんてできないけどね。多分、より初等的な証明ってのは無い、と思う。その関係上この定理はかなり応用的なものだとは思うのだけれど、AI曰く「事実としては既に知られており、確立された結果」だそう。まぁそんな簡単にフロンティアに手が届くとは思ってないけれど、名前も付かないくらい基礎的に扱われていると言われると中々効くものがある。さて、ザッと証明を記述しよう。

まず$\mathbb{F}^*_p$について考える。ただし、$\mathbb{F}^*_p$は$\mathbb{F}_p$の可逆元を集めた集合で、実体は$\mathbb{F}\backslash\{0\}$である。原始根の存在定理より、ある$r\in\mathbb{F}^*_p$が存在して全ての$\mathbb{F}^*_p$の元は$r$のべき乗で書ける。言い換えると、$\mathbb{F}^*_p$は積で以て$r$を生成元とする位数$p-1$の巡回群となる。したがって、加法群$\mathbb{Z}/(p-1)\mathbb{Z}$と同型である。この辺りは数論をやってたら自ずと出会うくらいの有名な事実である。
さて、写像$\mathbb{F}^*_p\to\mathbb{F}^*_p$として$x\mapsto x^n$を考える（使わない事実だが、群準同型である）。この写像は$\mathbb{F}^*_p$を加法群$\mathbb{Z}/(p-1)\mathbb{Z}$とみなすと$n$倍写像に対応する。つまり$x\in\mathbb{Z}/(p-1)\mathbb{Z}$上の写像$x\mapsto nx$であると解釈できる。したがって、$\mathbb{F}_p$における条件式$x^n=1$は$\mathbb{Z}/(p-1)\mathbb{Z}$においては$nx=0$だと見做せる。これを満たす$x$がちょうど$m$個あるということは、$n$倍写像の核の位数である$\gcd(n,p-1)$が$m$となることを意味する…と言って伝わらない可能性もあるのでもう少し初等的に書き下す。$\mathbb{Z}/(p-1)\mathbb{Z}$における$nx=0$の解がちょうど$m$個になるということは、$0,n,2n,\ldots,(p-2)n$の中に$p-1$で割り切れるもの（$p-1$の倍数）が$m$個存在するということである。一般に自然数$k$に対して$nk$が$p-1$の倍数になるということは、$n$と$k$が持つ素因数の指数を合算すると$p-1$のそれを超えるということを意味する。つまり$n$の素因数の指数の中でまだ$p-1$のそれに足りていない箇所を$k$をかけることで補完できたらOKである。さらに言い換えると、$n$が既に持っている$p-1$の因数成分は$n$と$p-1$の最大の共通因数（つまり$\gcd(n,p-1)$）であり、$k$が残りを補完するために持たなくてはならない、$n$が持っていなかった$p-1$の因数成分は$(p-1)/\gcd(n,p-1)$と書ける。つまり最終的には、$nx=0$の解は$(p-1)/\gcd(n,p-1)$の倍数であり、$0,1,2,\ldots,p-2$の中で$(p-1)/\gcd(n,p-1)$の倍数は$\gcd(n,p-1)$個である。
以上より、以下の主張が成り立つ。

- $m,n\in\mathbb{N}$として、素体$\mathbb{F}_p$に$x^n=1$の解がちょうど$m$個存在するような素数$p$が存在する必要十分条件は、$\gcd(n,p-1)=m$となるような素数$p$が存在することである。

したがって$m$が$n$の約数ではない場合は直ちにその様な素数$p$が存在しないことが分かる。なので$m$は$n$の約数としてしまっても良いだろう。以降ではそのように仮定して進める。

ここでディリクレの算術級数定理を用いる。定理によれば互いに素な自然数$A,B\in\mathbb{N}$に対して、$Ax+B$の形をした素数は無数に存在する。今、$\gcd(n/m,k)=1$を満たすように$k$を選ぶため、ある自然数$k_0$を初項とし、$n/m$を公差とする等差数列$k=k_0+(n/m)\ell$（$\ell\ge0$）を考える。このとき$\gcd(k_0,n/m)=1$であれば、任意の$\ell$について$\gcd(k,n/m)=1$が保証される。この$k$を用いて$p=km+1$を表すと以下の様になる。

$$
p=(k_0+(n/m)\ell)m+1=n\ell+k_0m+1
$$

これは初項$k_0 m + 1$、公差$n$の等差数列である。
ディリクレの算術級数定理より、この数列の中に素数が存在するための必要十分条件は、初項と公差が互いに素であること、すなわち$\gcd(k_0 m + 1, n) = 1$である。
ここで、$\gcd(k_0 m + 1, m) = 1$は常に成り立つため、$\gcd(k_0 m + 1, n) = 1$は$\gcd(k_0 m + 1, n/m) = 1$と同値になる。
よって、条件を満たす素数$p$が存在することは、「$\gcd(k_0, n/m) = 1$かつ$\gcd(k_0 m + 1, n/m) = 1$」を満たす自然数$k_0$が存在することと同値である。
これはまとめて$\gcd(k_0(k_0 m + 1), n/m) = 1$と表すことができる。
この条件を満たす自然数が存在することは、$m$と$n$の偶奇が一致することと同値である。

以上より、以下の主張が成り立つ。

- $n\in\mathbb{N}$およびその約数$m$に対して、$\gcd(n,p-1)=m$となるような素数$p$が存在する必要十分条件は、$m$と$n$の偶奇が一致することである。

条件$\gcd(k(km+1), n/m) = 1$を満たす自然数$k$が存在するかどうかを調べる。
$N = n/m$とおく。この条件は、$N$の任意の素因数$q$に対して、$k \not\equiv 0 \pmod{q}$かつ$km+1 \not\equiv 0 \pmod{q}$となる$k$が存在することと同値である。
中国剰余定理により、各素因数$q$に対して個別に条件を満たす剰余類が存在すれば、それらを連立させることで全体の条件を満たす$k$を構成できる。まず、$q$が奇素数の場合を考える。法$q$における剰余類は$3$つ以上存在する。条件によって禁止される剰余類は$k \equiv 0 \pmod{q}$と、$m \not\equiv 0 \pmod{q}$の場合における$k \equiv -m^{-1} \pmod{q}$の高々$2$つである。
したがって、奇素数$q$に対しては、禁止されない剰余類が必ず$1$つ以上存在するため、常に条件を満たす$k$を選ぶことができる。次に、$q = 2$の場合を考える。これは$N$が偶数である場合にのみ制約となる。$k \not\equiv 0 \pmod{2}$より、$k$は奇数（$k \equiv 1 \pmod{2}$）でなければならない。
このとき、$km+1 \equiv m+1 \pmod{2}$となる。
これが$2$の倍数とならないためには、$m+1 \equiv 1 \pmod{2}$、すなわち$m$が偶数でなければならない。逆に、$m$が偶数であれば、$k=1$を選ぶことで$k(km+1) = m+1$は奇数となり、条件を満たす。以上の議論から、条件を満たす$k$が存在しないのは「$n/m$が偶数であり、かつ$m$が奇数である」場合に限られる。$m$が$n$の約数であるという前提のもとでは、以下の通りとなる。$n$と$m$の偶奇が一致しない場合、$n$が偶数で$m$が奇数となるため、$n/m$は偶数となる。このとき上記の通り条件を満たす$k$は存在しない。$n$と$m$の偶奇が一致する場合、共に奇数であれば$n/m$も奇数となり$q=2$の条件は考慮不要となる。共に偶数であれば$m$が偶数であるため、先述の通り$q=2$の条件をクリアできる。

以上より、以下の主張が成り立つ。

- $n\in\mathbb{N}$およびその約数$m$に対して、$k(km+1)$が$n/m$と互いに素となる自然数$k$が存在する必要十分条件は、$m$と$n$の偶奇が一致することである。
